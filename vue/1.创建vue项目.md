创建vue2.x项目
=====
1.安装vue插件以及vue-cli
$ npm install -g vue
$ npm install -g vue-cli
2.创建vue项目：
$ vue init webpack [projectName]
3.安装依赖：
$ cd [projectName]
$ npm install
4.部署执行：
$ npm run dev
5.根据提示信息，查看监听端口，并在浏览器中打开


创建vue3.x项目
=====
1.安装vue插件以及vue-cli
$ npm install -g @vue/cli
2.创建vue项目：
$ vue create [projectName]
3.部署执行：
$ npm run serve
5.根据提示信息，查看监听端口，并在浏览器中打开
